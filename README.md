# .github

GitHub organization meta-repository for **QQ Omega**.

This repo holds the shared configuration and public profile for the [QQ Omega GitHub organization](https://github.com/QQomega-labs).

## Contents

```
.github/
├── img/                 # Shared images (org logo, social cards)
├── profile/
│   └── README.md        # Organization profile shown on GitHub
└── README.md            # This file
```

## What lives here

| Path                | Purpose                                                               |
| ------------------- | --------------------------------------------------------------------- |
| `profile/README.md` | Public-facing org profile rendered at `github.com/QQomega-labs`       |
| `img/`              | Static assets referenced by the profile and other org-level resources |
